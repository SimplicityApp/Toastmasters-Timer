Toastmaster Timer Documentation

Complete guide for adding, using, and removing Toastmaster Timer in Zoom. This app helps Toastmasters manage meeting timings, track speakers, and generate timing reports.

1. Adding the App to Zoom

Prerequisites

  • Zoom client version 5.13.0 or higher
  • Administrator access to install apps (for organization-wide installation)
  • An active Zoom account

Installation Methods

Method 1: Install from Zoom Marketplace (Recommended)

  1. Log in to your Zoom account at zoom.us
  2. Navigate to Zoom Marketplace (Apps → Zoom Marketplace)
  3. Search for "Toastmaster Timer"
  4. Click on the app listing
  5. Click the Install or Add button
  6. Review and accept the required permissions:
    • Virtual Background Control: Allows the app to change your virtual background to indicate timing phases (green, yellow, red)
    • Participant Information: Allows the app to access participant names for speaker selection
  7. Authorize the app installation
  8. The app will now be available in your Zoom client

Method 2: Developer Mode Installation (For Testing)

This method is for developers or users testing the app before it's published:

  1. Open your Zoom client
  2. Go to Settings → Apps → Developer Mode
  3. Enable Developer Mode
  4. Navigate to Zoom Marketplace → Your Apps → Local Test
  5. Click Add or Upload
  6. Upload the manifest.json file from the Toastmaster Timer app
  7. Grant the required permissions when prompted
  8. The app will be available for testing in your Zoom client
Note: After installation, you may need to restart your Zoom client for the app to appear in the Apps menu.

Verifying Installation

To verify the app is installed correctly:

  1. Start or join a Zoom meeting
  2. Click on Apps in the meeting toolbar
  3. Look for Toastmaster Timer in the list of available apps
  4. Click on it to launch the app

2. Using the App

Launching the App

  1. Start or join a Zoom meeting
  2. Click Apps in the meeting toolbar (bottom of the screen)
  3. Select Toastmaster Timer from the apps list
  4. The app window will open, displaying the timer interface

Main Features

Live Timer Tab

Start, pause, and manage timers for individual speakers with automatic virtual background changes.

Agenda Tab

Manage multiple speakers, import agendas, and organize your meeting schedule.

Report Tab

View and export timing reports for all completed speeches.

Using the Timer

Basic Timer Operation

  1. Select a Timing Role: Choose from predefined roles (Standard Speech, Ice Breaker, Table Topics, etc.) or select "Custom" to set your own timing thresholds
  2. Enter Speaker Name (Optional):
    • Type a name manually, or
    • Click the dropdown to select from Zoom meeting participants
  3. Start the Timer: Click the START button to begin timing
  4. Monitor Timing: The timer will automatically change your virtual background:
    • Green: Within the green zone (good timing)
    • Yellow: Entered the yellow zone (warning)
    • Red: Exceeded the red threshold (over time)
  5. Control the Timer:
    • STOP: Pause the timer
    • RESET: Clear the timer and start over
    • FINISH: Save the timing result to the report

Managing Multiple Speakers (Agenda Tab)

  1. Click the AGENDA tab
  2. Add Speakers:
    • Click "Add Speaker" to add manually
    • Or use "Import from Text" to paste a list of names
    • Or use "Import from Easy-Speak" if you use Easy-Speak for meeting management
  3. Reorder Speakers: Drag and drop speakers to reorder them
  4. Load Speaker: Click on a speaker's name to load them into the timer
  5. Mark Complete: Speakers with completed timings will show a checkmark

Customizing Timing Rules

  1. In the Live Timer tab, click "Edit timing rules" next to the role dropdown
  2. Modify the green, yellow, and red thresholds for each speech type
  3. Click Save to apply changes
  4. Your custom rules are saved locally and will persist across sessions

Viewing and Exporting Reports

  1. Click the REPORT tab
  2. View all completed timings with speaker names, roles, and timing results
  3. Click "Copy Report to Clipboard" to export
  4. Paste the report into Excel, Google Sheets, or any text editor
  5. The report is formatted as tab-separated values for easy import

Virtual Background Feature

How it works: The app automatically changes your Zoom virtual background to indicate timing phases:

  • Green background: Speaker is within the acceptable time range
  • Yellow background: Speaker has entered the warning zone
  • Red background: Speaker has exceeded the time limit

Requirements: You must grant virtual background permissions during installation for this feature to work.

Tips for Best Results

  • Ensure you're in a Zoom meeting before launching the app
  • Grant all requested permissions during installation for full functionality
  • Use the Agenda tab to prepare your speaker list before the meeting starts
  • Customize timing rules to match your club's specific requirements
  • Export reports immediately after meetings to avoid data loss

3. Removing the App

Uninstalling from Zoom

Method 1: Remove via Zoom Web Portal

  1. Log in to your Zoom account at zoom.us
  2. Navigate to Zoom Marketplace
  3. Click on Manage → Installed Apps (or go to My Apps)
  4. Find Toastmaster Timer in the list of installed apps
  5. Click on the app
  6. Click the Uninstall or Remove button
  7. Confirm the uninstallation when prompted
  8. The app will be removed from your Zoom account

Method 2: Remove via Zoom Client

  1. Open your Zoom client
  2. Go to Settings → Apps
  3. Find Toastmaster Timer in the list of installed apps
  4. Click on the app
  5. Select Uninstall or Remove
  6. Confirm the removal

Clearing Local Data

After uninstalling the app, you may want to clear any locally stored data:

  1. Open your web browser
  2. Access browser settings/preferences
  3. Navigate to Privacy or Site Settings
  4. Find Storage or Local Storage
  5. Search for the Toastmaster Timer domain
  6. Clear the stored data (this will remove your agenda, reports, and custom timing rules)
Important: Clearing local data will permanently delete all your saved agendas, timing reports, and custom timing rules. Make sure to export any important reports before clearing data.

Reinstalling the App

If you want to reinstall the app after removal, simply follow the installation instructions in Section 1 above. Note that any previously stored local data will be lost unless you've backed it up.

4. Troubleshooting

Common Issues

Issue Solution
App doesn't appear in Apps menu Restart Zoom client, verify installation in Zoom Marketplace, check that you're in a meeting
Virtual background not changing Verify permissions were granted, ensure you're in a meeting, check Zoom client version (5.13.0+)
Participant list is empty Check Zoom API permissions, ensure you're in a meeting with participants, try refreshing the app
Data not saving Check browser localStorage permissions, ensure cookies/local storage is enabled
App crashes or freezes Close and reopen the app, refresh the Zoom client, check for Zoom client updates
