# Toastmaster Timer - Video Production Guide (Zoom App)

Step-by-step instructions to create the product demo video using AI tools.

## PHASE 1: Preparation (1-2 hours)

### Step 1: Set Up a Demo Environment

1. Open Zoom and start a meeting by yourself (or with 1-2 friends to simulate participants)
2. Open the Toastmaster Timer Zoom app in the sidebar
3. Pre-load a realistic agenda with 5-6 speakers:
   ```
   Sarah Chen (Ice Breaker)
   James Wilson (Standard Speech)
   Maria Garcia (Table Topics Speech)
   David Kim (Speech Evaluation)
   Emily Brown (General Evaluation)
   Alex Thompson (Short Roles)
   ```
4. Pre-run 2-3 speeches to completion so the Report tab has data to show
5. Make sure your Zoom window is clean — close other apps, hide bookmarks bar, use a simple desktop wallpaper

### Step 2: Prepare the Script

1. Open `video-script-zoom-app.md` in this repo — it has the full narration script
2. Read through it and adjust any wording to match your voice/style
3. Copy each scene's narration text — you'll paste these into HeyGen later

### Step 3: Plan Your Recordings

You'll record **6 separate screen clips** (one per product scene). Keeping them separate makes editing much easier.

| Clip # | Scene | What to show | Duration |
|--------|-------|-------------|----------|
| 1 | Scene 2 | Open Zoom, show app in sidebar, Live tab overview | 10-15 sec |
| 2 | Scene 3a | Select role, type speaker name with autocomplete | 10-15 sec |
| 3 | Scene 3b | Start timer, show color changes, toggle overlay modes | 20-25 sec |
| 4 | Scene 4 | Click Finish, auto-advance to next speaker | 10-15 sec |
| 5 | Scene 5 | Switch to Agenda tab, import text, drag-and-drop | 25-30 sec |
| 6 | Scene 6 | Switch to Report tab, show table, copy to clipboard | 15-20 sec |
| 7 | Scene 7 | Open Edit Rules modal, modify a rule, save | 10-15 sec |

## PHASE 2: Screen Recording (1-2 hours)

### Option A: Screen Studio (Recommended — Mac)

Screen Studio auto-adds zoom effects when you click, making recordings look polished without manual editing.

1. Download Screen Studio from [screen.studio](https://screen.studio/)
2. Open Screen Studio, select "New Recording"
3. Set recording area to your **Zoom window only** (not full screen)
4. In Screen Studio settings:
   - Resolution: 1920x1080
   - Auto zoom: ON (it will zoom in when you click UI elements)
   - Cursor: Smooth tracking ON
   - Background: Choose a subtle gradient or solid color
   - Padding: Add slight padding around the Zoom window for a clean look
5. Record each clip separately:
   - Click Record, wait 2 seconds, perform the actions slowly and deliberately
   - Pause between clicks so the auto-zoom has time to animate
   - Click Stop when done
6. Export each clip as MP4 (1080p, 60fps)

### Option B: QuickTime (Free — Mac)

1. Open QuickTime Player > File > New Screen Recording
2. Select "Record Selected Portion" and drag over your Zoom window
3. Click Record, perform the actions, click Stop
4. Save each clip as a .mov file
5. Note: no auto-zoom — you'll add zoom effects in Descript or CapCut later

### Option C: HeyGen Built-in Screen Recorder (Free)

1. Go to [heygen.com](https://www.heygen.com/) and sign up (free)
2. Click "Screen Recorder" from the dashboard
3. Select the Zoom window to record
4. Record each clip — HeyGen will auto-clean audio and add captions
5. Export clips to use in the next phase

### Screen Recording Tips

- **Go slow.** Move your cursor deliberately. Viewers need time to see what you're clicking.
- **For the color transitions (Clip 3):** Use "Table Topics Speech" role (Green: 1:00, Yellow: 1:30, Red: 2:00) so you don't have to wait 5+ minutes. You'll speed this up 4-6x in editing.
- **Clean up notifications.** Turn on Do Not Disturb before recording.
- **Reset between clips.** Clear the timer, go back to a clean state before each clip.
- **Record each clip 2-3 times.** Pick the best take later.

## PHASE 3: AI Avatar Presenter (1-2 hours)

You'll generate avatar clips that introduce each scene. These get interleaved with the screen recordings.

### Using HeyGen

1. **Sign up** at [heygen.com](https://www.heygen.com/) (free tier: 3 videos/month, 720p)
2. **Create your avatar:**
   - Option A: Pick a stock avatar from HeyGen's library (230+ options)
   - Option B: Upload a photo of yourself to create a custom "Photo Avatar"
   - Option C (paid): Record a 2-min training video to create a realistic clone of yourself
3. **Create presenter clips** — make one video per scene that needs narration:

   **Video 1: Hook / Problem (Scene 1)**
   - Click "Create Video" > "AI Avatar"
   - Select your avatar
   - Paste the narration: *"If you've ever been the Timer in a Toastmasters meeting on Zoom, you know the struggle..."*
   - Choose voice (or clone your own voice on paid plan)
   - Background: transparent or solid color (you'll overlay this in editing)
   - Generate → Download

   **Video 2: Product Intro (Scene 2)**
   - Narration: *"Meet Toastmaster Timer — a Zoom app that runs right inside your meeting..."*
   - Generate → Download

   **Video 3: Live Timer narration (Scene 3)**
   - Narration: *"Select a role — like Standard Speech — and the app loads the correct timing rules... Hit Start, and the timer counts up..."*
   - Generate → Download

   **Video 4: Finish & Advance (Scene 4)**
   - Narration: *"When the speech ends, hit Finish. The app records the duration..."*
   - Generate → Download

   **Video 5: Agenda (Scene 5)**
   - Narration: *"Before the meeting, set up your agenda in seconds..."*
   - Generate → Download

   **Video 6: Reports (Scene 6)**
   - Narration: *"After the meeting, your report is ready..."*
   - Generate → Download

   **Video 7: Customization (Scene 7)**
   - Narration: *"Every role comes with standard Toastmasters timing rules..."*
   - Generate → Download

   **Video 8: CTA (Scene 8)**
   - Narration: *"Toastmaster Timer takes the stress out of the Timer role..."*
   - Generate → Download

4. **Download all clips** as MP4

### Tips for HeyGen

- Free tier gives you 3 videos/month — you may need the $29/mo Creator plan for all 8 clips
- Alternatively, combine narrations into fewer videos (e.g., merge Scenes 3-4 into one clip)
- Choose a **professional but friendly** voice tone
- Use a **transparent or green screen background** so you can overlay the avatar as a small circle in the corner of your screen recordings

## PHASE 4: Editing & Assembly (2-3 hours)

This is where everything comes together. Use **Descript** (recommended) or **CapCut**.

### Option A: Descript (Recommended)

Descript lets you edit video by editing text — perfect for narration-heavy videos.

1. **Sign up** at [descript.com](https://www.descript.com/) (free: 60 min/month, 720p)
2. **Create a new project** called "Toastmaster Timer - Zoom App Demo"
3. **Import all your files:**
   - Drag in all screen recording clips (Clips 1-7)
   - Drag in all HeyGen avatar clips (Videos 1-8)
4. **Build the timeline scene by scene:**

   **Scene 1 — The Problem (0:00 - 0:15)**
   - Place the avatar Video 1 (hook narration) as the main track
   - If you have stock footage of someone frustrated on Zoom, overlay it
   - Otherwise, just use the avatar talking — it works fine

   **Scene 2 — Product Intro (0:15 - 0:30)**
   - Start with avatar Video 2 speaking
   - At "runs right inside your meeting," cut to screen recording Clip 1 (Zoom with app in sidebar)
   - Optionally: picture-in-picture — small avatar circle in bottom-left, screen recording fills the frame

   **Scenes 3-7 — Feature Demos**
   - For each scene, use this pattern:
     1. Avatar narration plays as voiceover (audio track)
     2. Screen recording plays as the visual (video track)
     3. The avatar speaks, and the viewer sees the app in action simultaneously
   - In Descript, this means: place the screen recording clip on the timeline, then drag the avatar audio on top as a separate track

   **Scene 8 — CTA (2:30 - 2:50)**
   - Avatar Video 8 full screen, with logo and marketplace URL overlaid as text

5. **Edit the transcript:**
   - Descript auto-transcribes everything
   - Delete any "um"s, pauses, or awkward gaps by deleting words in the text
   - The video cuts automatically
6. **Add captions:**
   - Click "Captions" > "Auto-generate"
   - Choose a style (white text with dark background works well)
7. **Add transitions:**
   - Simple crossfade (0.3-0.5 sec) between scenes
8. **Add background music:**
   - Descript has a built-in royalty-free music library
   - Choose something upbeat but subtle, lower the volume to ~15-20%
9. **Export:**
   - File > Export > MP4
   - Resolution: 1080p (paid) or 720p (free)
   - Upload to YouTube, your website, or social media

### Option B: CapCut (Free alternative)

1. **Open** [capcut.com](https://www.capcut.com/) or download the desktop app
2. **Create a new project** (16:9 aspect ratio)
3. **Import** all screen recordings and avatar clips
4. **Drag clips onto the timeline** in scene order:
   - Main track: screen recordings
   - Overlay track: avatar clips (resize to small circle in corner)
   - Audio track: avatar narration audio
5. **Add auto-captions:**
   - Click "Text" > "Auto Captions" > select language > Generate
   - Pick a caption style template
6. **Add text overlays** for feature callouts:
   - "Auto Background Change" (Scene 3)
   - "One-Click Import" (Scene 5)
   - "Instant Reports" (Scene 6)
7. **Add transitions** between scenes (use "Fade" or "Slide")
8. **Add music** from CapCut's built-in library
9. **Export** at 1080p

## PHASE 5: Final Polish (30 min - 1 hour)

### Add Text Overlays & Callouts

Add these brief text labels when each feature first appears:

| Timestamp | Text Overlay |
|-----------|-------------|
| 0:30 | "Auto Timing Rules" |
| 0:45 | "Auto Background Colors" |
| 1:05 | "Timer Card / Timer + Camera modes" |
| 1:15 | "Auto-Advance to Next Speaker" |
| 1:30 | "One-Click Agenda Import" |
| 1:55 | "Instant Reports" |
| 2:15 | "Fully Customizable Rules" |

### Add Logo & Branding

- Add the Toastmaster Timer logo watermark in the top-right corner (small, ~10% opacity)
- Use consistent colors from the app (blues, greens)
- Final frame: logo centered, marketplace URL below, "Also available as a web app" text

### Add an Intro Card (Optional)

- 3-second title card at the very start:
  - App logo
  - "Toastmaster Timer"
  - "Run the Timer role easily"

### Review Checklist

- [ ] Audio levels consistent across all clips (narration at same volume)
- [ ] No dead air or awkward pauses longer than 1 second
- [ ] Captions are accurate (fix any auto-caption errors)
- [ ] All screen recordings are sharp and readable
- [ ] Background music doesn't overpower narration
- [ ] Total length is between 2:30 and 3:00
- [ ] CTA is clear with the Zoom Marketplace URL visible
